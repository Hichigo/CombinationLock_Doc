# BP\_SymbolsLock

It works the same with numbers, they go clockwise, only instead of numbers on the texture there are symbols. Also added logic in which after entering the correct password, the symbols start to glow.

## Settings

### Lock

| **Variable name** | **Type** | **Description**                                                                        |
| ----------------- | -------- | -------------------------------------------------------------------------------------- |
| **PowerEmissive** | float    | The strength of the emission of symbols, displayed after entering the correct password |
