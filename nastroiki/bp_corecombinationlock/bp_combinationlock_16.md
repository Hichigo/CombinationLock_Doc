# BP\_CombinationLock\_16

This blueprint inherits from **BP\_CombinationLock**, it is configured in it:

* **NumberOfDigitsInLock -** set to 16
* **BigStepDigitRotation -** set to 5
* **PasswordCombination -** the combination is configured
* **SM\_CombinationLockHand -** changed texture with numbers
